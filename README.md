# preprocessing_sequencing_data

Welcome to the tutorial for preprocessing sequencing data component of the workshop! 

Use the left sidebar to browse files, and click on them to open them in the file viewer. The following dependencies should be pre-installed in your terminal:

- samtools
- minimap2
- ivar

You'll find the steps as directories in the left sidebar, starting with Step1. These contain all the instructions and data needed for each step of the tutorial. Go ahead and cd into Step1 for the first step, and take a look at Step1.md by clicking on the file in the sidebar.

Steps 1-3 are for processing amplicon-based sequencing data.
Have fun!
